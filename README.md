# Temple of the King

## Development

```sh
# Install the correct node version
nvm install 20.16.0

# If already installed make sure you are using it
nvm use 20.16.0

# Install dependencies
npm install

# Start the server
npm start
```

## Build

```sh
npm run build
```

Build output lives in the repo (`docs` folder) to make use of deployment of GitHub pages.
